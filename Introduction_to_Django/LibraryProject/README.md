# LibraryProject

This is a Django project created as part of the "Introduction to Django" learning module. The project demonstrates basic Django concepts including:

- Django project setup and configuration
- Creating Django apps
- Working with Django models and ORM
- Using the Django admin interface

## Project Structure

- `LibraryProject/` - Main project directory containing settings and configuration
- `bookshelf/` - Django app for managing books
- `manage.py` - Django command-line utility

## Getting Started

1. Navigate to the project directory
2. Run the development server: `python manage.py runserver`
3. Access the application at http://127.0.0.1:8000/

## Apps

### bookshelf
A Django app that manages a library's book collection with CRUD operations and admin interface integration.
